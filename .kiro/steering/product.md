# Product Overview

HackIndia 2026 - A landing page for India's biggest Web3 & AI hackathon event.

## Purpose
Single-page marketing website showcasing event details, prizes, sponsors, themes, rules, and coordinator information for a 24-hour hackathon at Mailam Engineering College (March 13-14, 2026).

## Key Features
- Event countdown timer
- Event information sections (About, Prizes, Sponsors, Themes/Rules, Coordinators)
- Responsive design with dark theme
- Animated UI elements
- Registration and brochure download CTAs

## Target Audience
Students, developers, and tech enthusiasts interested in Web3 and AI hackathons.
